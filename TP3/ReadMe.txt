Pour jouer au jeu, il faut lancer "main.py".

La liste des scores, du meilleur au pire s'affiche
dans la barre de menu.

À part le fond du canvas de jeu et le "boom" de
lorsqu'on tue un ennemi, toutes les images du
dossier image ont été faites main sur paint.

Lorsque vous quitter, vous pouvez sauvegarder votre
progression. Tout est expliqué directement dans
le jeu.

Le magasin d'amélioration n'est pas encore bridé
car je n'ai pas vraiment la notion de quelle vitesse
de tire est bien etc...
Pour le brider, il suffirait juste de faire une
vérification avec un "if" de combien d'amélioration
on déjà était faites (var up_)


Dans le data :
-> Niveaux :
ce dossier permet de gérer et même de créer ses
propres niveaux. Les ennemis sont marqués avec
un 1. En effet, le but est qu'il soit possible
dans le futur que par exemple on créé d'autre
classe d'ennemis de type 2, 3, etc... avec des
particularités.
